<h1>Course Management Software</h1>
<h2>Software Engineering Final Project (CSCI 4711)</h2>


<h3>Overview</h3>
<p>Designed and implemented a full-stack course management desktop application in Java after completing a semester-long software design project. While the original assignment focused only on system design, I independently built a working implementation based on our team’s RAD (check it out for more info - RAD Deliverable 4.pdf).</p>

<h3>Key Features</h3>
<p>
  
- Implemented secure authentication using SHA-256 hashed passwords
- Developed role-based functionality for students and teachers
- Teachers can input/edit student grades
- Students can browse courses and register for sections
- Integrated SQLite database for persistent data storage
- Designed a responsive UI using JavaFX, including scene and layout management
- Ensured real-time synchronization between UI actions and database updates</p>

<h3>Technical Approach</h3>
<p>

- Implemented a layered MVC architecture incorporating boundary, control, and entity design patterns
- Translated UML diagrams (sequence diagrams, object models) into working code
- Extended original designs by introducing additional classes and methods to improve communication between system components
- Maintained separation of concerns by restricting database access to controller-level logic
</p>

<h3>Challenges and Learning</h3>
<p>

- Bridging the gap between theoretical design artifacts and real-world implementation
- Adapting rigid UML designs to accommodate practical development needs
- Learning JavaFX and effectively implementing UI layouts and navigation
- Recognizing the importance of flexibility in design and iterative refinement during development
</p>

<h3>Outcome</h3>
<p>Successfully delivered a fully functional application beyond the original course requirements, demonstrating the ability to independently carry a project from design to implementation.</p>
